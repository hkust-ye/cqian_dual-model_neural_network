data of cantilever beams, generated by density-based topology optimization and used to train the dual-model neural network

dual-model_neural_netowor_for_compliance_problem.py: code of dual-model neural network for minimal compliance problem

dens.npy: (dens_part1.npy+dens_part2.npy) structural density distribution

fy.npy: (fy_part1.npy+fy_part2.npy) the loading/force distribution applied on the structure

Compliance.npy: Structural compliance value

comp_sen.npy: (comp_sen_part1.npy+comp_sen_part2.npy) the structural complianace's sensitivity with respesct to the density distribution.

num.txt: the number of iterations for each topology optimization case. There are 126 cases in total.
