data of cantilever beams, generated by density-based topology optimization and used to train dual-model neural network

dens.npy: structural density distribution

fy.npy: the loading/force distribution applied on the structure

Compliance.npy: Structural compliance value

comp_sen.npy: the structural complianace's sensitivity with respesct to the density distribution.

num.txt: the number of iterations for each topology optimization case. There are 126 cases in total.
