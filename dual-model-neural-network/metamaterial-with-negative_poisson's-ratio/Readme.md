data of metamaterials with negative Poisson's ratio, generated by density-based topology optimization and used to train the dual-model neural network

densi.npy: (densi_part1.npy+densi_part2.npy) structural density distribution

obj.npy: Three components in the structural effective stiffness tensor

sensi1.npy: (sensi1_part1.npy+sensi1_part2.npy) the sensitivity of the first component in obj.npy with respesct to the density distribution.

sensi2.npy: (sensi2_part1.npy+sensi2_part2.npy) the sensitivity of the second component in obj.npy with respesct to the density distribution.

sensi3.npy: (sensi3_part1.npy+sensi3_part2.npy) the sensitivity of the third component in obj.npy with respesct to the density distribution.

num.txt: the number of iterations for each topology optimization case. There are 104 cases in total.
